# Tanya Panchal - Portfolio Website

A professional, minimalist portfolio website built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- **Modern Design**: Clean, minimalist black and white aesthetic
- **Smooth Animations**: Lightweight Framer Motion animations with scroll-based reveals
- **Fully Responsive**: Optimized for all device sizes
- **SEO Optimized**: Built with Next.js best practices and Open Graph tags
- **Performance Focused**: Fast loading and smooth scrolling
- **Working Contact Form**: Functional contact form with API integration
- **CV Download**: Direct download functionality for resume
- **Navigation**: Fixed navigation with active section highlighting

## Tech Stack

- Next.js 14 with App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Inter Font

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
npm start
```
